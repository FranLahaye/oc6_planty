Un plugin simple pour personnaliser la palette de couleurs & tailles de texte 
& blocs de l'éditeur Gutenberg 


== Description ==

Ce plugin permet de **modifier la palette de couleurs par défaut et les tailles de texte**.
De plus cette extension permet de **personnaliser** l'éditeur et les blocs «Gutenberg».

Une fois le plugin installé et activé, accédez à ECCP Réglages pour personnaliser les couleurs.

== Fontionnalités ==
Ce plugin comprend 10 sections.

PALETTE DE COULEURS

◆ Personnaliser une palette de 7 à 12 couleurs:
Un réglage d'opacité permet si vous le souhaitez d'appliquer une couleur RGBA.
Vous pouvez définir un nom simple pour la couleur.
◆ Pour le thème OceanWP remplacer la palette de 8 couleurs par défaut dans le 
sélecteur de couleurs du personnaliseur par notre palette de couleurs personnalisées
◆ Option pour désactiver les couleurs personnalisées
◆ Option pour désactiver des dégradés personnalisés
◆ Compatible avec palette de couleurs définit par theme.json


TAILLES DE TEXTE DE L'ÉDITEUR

◆ Définir les tailles de texte : Petite, Normale, Large, Énorme.
◆ Option pour activer la typographie fluide pour les tailles de texte prédéfinies.
◆ Définir la taille de texte par défaut de l'editeur Gutenberg.
◆ Option pour désactiver la taille de police personnalisée.
◆ Option pour activer la typographie fluide pour les tailles personnalisées.
◆ Compatible avec tailles de texte définit par theme.json.


PERSONNALISATION DE L'ÉDITEUR GUTENBERG

◆ Option pour désactiver le mode plein écran de l'éditeur Gutenberg
◆ Option pour personnaliser la couleur d'arrière plan de l'editeur
◆ Option  pour activer la feuille de style par défaut des blocs Gutenberg sur Front End
◆ Option pour activer l’alignement large ou pleine largeur pour l’éditeur de blocs(par exemple bloc image)
◆ Option pour activer les embeds responsives(par exemple intégration de vidéos YouTube) 
◆ Option largeur max des vidéos avec les blocs contenus embarqués(Embeds)
◆ Largeurs maximales de l'éditeur, faites correspondre la largeur de l'éditeur à la largeur de vos sites.
◆ Option d'alignement pour le titre des publications dans l'éditeur
◆ Option pour activer la prise en charge des nouveaux 
paramètres de bloc(Marge,Bordure ...)introduit depuis WordPress 6.0 
◆ Option désactiver pour les images JPEG la génération de  miniatures au format WebP introduit par WordPress 6.1
◆ Option désactiver le téléchargement des images WebP & pour les images JPEG la génération de miniatures au format WebP 
◆ Option pour désactiver la catégorie Openverse de l'onglet Médias
◆ Option qui permet de désactiver pour tous les blocs les sous-panneaux paramètres & Styles 
afin d'afficher les paramètres et styles du bloc dans le seul panneau bloc de la barre latérale de l'editeur.
◆ Options pour régler des espacements fluides qui s'appliquent aux sept valeurs prédéfinies des marges interne, 
externe et espacement des blocs qui sont disponibles dans la section Dimensions de l'onglet Style de l'editeur de blocs.

PERSONNALISATION DES BLOCS GUTENBERG

◆ Personnaliser le style des blocs Paragraphe,Liste,Image,Boutons,Colonne,Couverture,
Détails,Galerie,Media/texte,Lire la suite,Groupe,Separateur,Titre,Tableau,
Derniers Articles,Boucle De Requête,Espacement,FAQ Yoast SEO,Icônes de réseaux sociaux

◆ Exemple de personnalisation:
- Transformer en Front End le bloc Liste en accordéon par exemple pour établir une FAQ
- Transformer en Front End le bloc Groupe pour établir une Table Des Matières
- Ajouter des formes de séparation au bloc Groupe
- Transformer en Front End le bloc galerie par un Slider
- Transformer en Front End le bloc galerie composée de deux images qui présentent des 
différences pour une comparaison Avant/Aprés   


ADDONS BLOCS RÉUTILISABLES

◆ Option pour activer un menu des blocs réutilisables en back office
◆ Option pour ajouter des shortcodes et des fonctions 
PHP à l'interface d'administration des blocs réutilisables.
◆ Option pour ajouter automatiquement du contenu sous forme de blocs réutilisables avant 
et/ou après vos publications, articles, pages ou tout type de publication personnalisée.
◆ Vous pouvez également empêcher votre contenu avant/après de s'afficher sur tous
type de publication via le Panneau Contenu avant/aprés publication de la barre latérale de 
Publications de l'editeur ainsi que sur l'edition rapide ou groupé pour tous les écrans qui 
liste les publications.
Attention! pour utiliser cette fonctionnalité sur un type de publication personnalisé
vous devez activer le support champ personnalisé de votre CPT.
◆ Dans cette section vous pouvez télécharger des blocs réutilisables au format json:
Bloc Conseil,Bloc Information,Bloc Attention,Bloc Alerte,Blockquote,Bloc mise en évidence,
Table Des Prix,Boucle De Requête Articles,Bloc Icônes Sociales avec liens de partage,
Bloc Tables des Matières 


CONSTRUCTEUR DE PAGES ELEMENTOR

◆ synchroniser la palette de couleurs Elementor avec la palette de couleurs Gutenberg


THÈME DIVI

◆ synchroniser la palette de couleurs DIVI avec la palette 
de couleurs Gutenberg ou réciproquemment la palette de 
couleurs Gutenberg avec la palette de couleurs DIVI


FORMATS DE TEXTE ENRICHI

**Editor Custom Color Palette** ajoute également des 
**formats de texte manquants** pour une meilleure édition de contenu.

◆ Surlignement du texte selon une palette de 7 couleurs ou couleur personnalisée
◆ Soulignement simple du texte 
◆ Soulignement du texte selon une palette de 7 couleurs ou couleur personnalisée
◆ Lien Externe: Permet l’ajout d’une icône Dashicons External après le lien sélectionné 
avec réglage de la couleur du lien + icône selon une palette de 7 couleurs ou couleur personnalisée.
Cela met en évidence que c'est un lien externe qui 
redirigera l'utilisateur vers un autre site Web.
◆ Encadrement du texte avec un contour de couleurs au trait simple,double,pointillé.
◆ Afficher une annotation dans une info-bulle.
◆ Ajout d'un contôle Clear Formatting pour le menu format de la barre d'outils du bloc.
Ce bouton supprime tous les formats actif du texte sélectionné(gras, italique etc...)


VARIATION DE BLOCS

A partir de **l'inséreuse de blocs** sous la catégorie **BLOCS PERSONNALISÉS ECCP** 
vous disposez des variations de blocs suivante:

◆ Info Box
◆ Click to Tweet
◆ Pricing Table
◆ Fancy Quote (variation du bloc natif de Gutenberg: citation)
◆ Countdown Timer (compte à rebours)
◆ Flip Box (boite à retournement)


VISIBILITÉ DES BLOCS

◆ Afficher ou masquer les blocs Gutenberg sur les ordinateurs de bureau, les tablettes et 
les mobiles.


PERSONNALISER BACK OFFICE

◆ Ajout d'une colonne pour afficher une vignette de l'image en avant pour les écrans de
publications qui répertorient les Articles, Pages, Custom Post Type
◆ Ajout d'un réglage à l'écran d'édition rapide qui permet de supprimer/ajouter une image 
en avant 
◆ Ajout d'une colonne pour afficher l'ID de publication pour les écrans qui répertorient 
les Articles, Pages, Custom Post Type
◆ Rendre la colonne triable pour les Taxonomies des types de publication personnalisé
◆ Filtre par auteur pour tous les types de publication
◆ Filtre par catégories pour les types de publication personnalisé
◆ Filtre par plage de date pour les articles & types de publication personnalisé & médiathèque
◆ Filtre pour les articles par statuts de publication(Brouillon, En attente de relecture etc..)
◆ Option status de publication pour les actions groupés des articles
◆ Ajout des colonnes a la médiathèque:ID de publication,Dimensions: largeur * Hauteur,
Taille du fichier en Ko
◆ Autoriser le téléchargement de médias au format SVG et ajout du type SVG pour le filtre
par type de fichier de la médiathèque.
◆ Définir automatiquement le texte alternatif et la légende selon le titre de l'image lors 
du téléchargement.
◆ Ajout d'une colonne sur l'écran tous les comptes pour afficher la date de dernière connexion
◆ Ajout d'une colonne sur l'écran tous les comptes pour afficher la date d'enregistrement des utilisateurs
◆ Option pour autoriser le téléchargement des médias dans la médiathèque pour les utilisateurs qui ont un rôle de contributeur. 
◆ Page de connexion: désactiver le sélecteur de langue,ajouter un message personnalisé 
d'erreur de connexion,désactiver la demande de vérification e-mail administrateur,activer un logo 
personnalisé,personnaliser le formulaire de connexion,ajouter une image de fond,rediriger aprés connexion les administrateurs
vers le tableau de bord et les autres utilisateurs vers la page d'accueil.
◆ EMAIL: Modification du nom de l'expéditeur et de l'adresse e-mail WordPress Par défaut pour les 
notifications d'administration.
Permettre que tous les e-mails sortants de WordPress utiliseront «l'adresse de retour» 
correspondant à l'adresse «De», ce qui peut aider à améliorer la délivrabilité des e-mails.
◆ Désactiver le sitemap de WordPress par défaut.
◆ Activer pour un site en local ou en ligne un item de menu sous forme d'une Icône Dashicons afin d'établir  
une connexion/déconnexion à l'admin de WordPress.
◆ Tableau de bord WordPress: ajout dans le widget "d'un coup d'oeil" du nombre de blocs réutilisables,
nombre d'utilisateurs,nombre par type de publications personnalisées.

PERSONNALISER FRONT END
◆ Options pour activer et personnaliser un bouton "retour en haut"
◆ Options pour activer et personnaliser une boite d'info auteur a la fin des articles
◆ Option pour activer la fenêtre contextuelle Fancybox pour apporter une lightbox aux images & galeries WordPress.
◆ Options pour personnaliser le formulaire de commentaire natif de WordPress

== Installation ==
1. Depuis le tableau de bord de votre site, accédez à Extensions -> Ajouter nouveau.
2. Rechercher Editor Custom Color Palette
3. Cliquez sur installez et activer le plugin
4. Accédez aux réglages du plugin via le menu **ECCP Réglages**.
